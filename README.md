This skill solves the following problem: AI conversations are suitable for immediate Q&A, but not for long-term learning around the same topic. After multiple follow-up questions, knowledge can easily pile up as chronological chat records and become fragmented, repetitive, and hard to review.

This skill is suitable for agents that can operate on files, such as Codex and Claude Code.

Do not be dragged down by sunk cost. If the AI has tried more than three times to explain something and still cannot help you understand it, ask it to create a new document and rewrite the explanation from scratch.

Using Git and VS Code is recommended, so it is easier to see where the AI made changes each time.
